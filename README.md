# Moododoro

Moododoro is a customizable Pomodoro timer designed to help you focus while matching your mood.

Instead of forcing you into a rigid productivity interface, Moododoro lets you shape the environment you work in — with customizable colors, backgrounds, and eventually even ambient video backgrounds.

The goal is simple: **focus better by creating a workspace that feels right.**

---

## Features

### Pomodoro Timer Engine

Moododoro implements a configurable Pomodoro workflow powered by a reducer-based state machine.

Users can customize:

- Work duration
- Short break duration
- Long break duration
- Long break interval

---

### Mood-Based Backgrounds

The app allows users to customize the background environment to match their focus mood.

Supported background types:

- Custom colors
- Image backgrounds
- (Planned) YouTube video backgrounds

The default relaxing background color is:

```
#51c4cc
```

---

### Minimal Focus Interface

Moododoro intentionally keeps the UI minimal so the timer stays the center of attention.

Key interface goals:

- Distraction-free layout
- Simple controls
- Clear timer visibility

---

If you have ideas for improving focus workflows, feel free to submit a feature request or open a pull request.
